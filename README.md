📌 Overview

Health-Synergy is a frontend prototype of a telehealth platform designed to improve healthcare accessibility, especially for rural and underserved communities. The project explores how digital platforms can provide health information, symptom guidance, and wellness recommendations.

This repository contains the website prototype built using HTML, CSS, and JavaScript.

🌐 Features

Informational telehealth website with multiple pages

Navigation system (Home, About, Contact)

User input form for collecting basic details

JavaScript-based interactivity and DOM manipulation

Conceptual design for telehealth accessibility

🛠 Tech Stack

HTML5 – Website structure

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Interactivity and form handling

🎯 Project Goals

Understand web development fundamentals

Explore digital healthcare platforms

Design solutions for rural healthcare accessibility

Learn frontend development and UI/UX basics

🌍 Social Impact Concept

The platform concept focuses on reducing healthcare barriers by enabling remote access to medical guidance, especially for users in low-connectivity regions.
